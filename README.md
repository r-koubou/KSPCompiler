KSPCompiler
===========

A compiler programs for KONTAKT Script Processor scripts.


## Features

- Compiler
    - Syntax analysis
    - Semantic analysis
    - Obfuscate
- Language server
- Built-in Symbol Management
    - Variable
    - UI Type
    - Command
    - Callback


## Requirements

- .net 9.0 Runtime
