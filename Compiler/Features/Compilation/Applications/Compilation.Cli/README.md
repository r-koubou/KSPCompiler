KSPCompiler
===========

A compiler for KONTAKT Script Processor scripts.

## Requirements

- .net 9.0 Runtime

## Usage

``` bash
kspc [options...] [-h|--help] [--version] <script_file>
```

* Options:
    * `--input` A script file path to compile. (Required)
    * `--enable-obfuscation`    Run obfuscation after compiling. (Optional)
